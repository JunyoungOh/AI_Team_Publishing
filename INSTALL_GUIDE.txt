====================================================================
  1. 이 프로그램이 하는 일
====================================================================

이 프로그램은 "AI 에이전트 팀"입니다.

여러분이 작업을 지시하면, AI가 아래처럼 역할을 나누어 일합니다:

  CEO (최고 관리자)
    -> Leader (팀장들)
       -> Worker (실무자들)

예를 들어 "카카오 주가 분석 보고서를 작성해줘"라고 지시하면:
  - CEO가 작업을 분석하고 필요한 팀을 구성합니다
  - Leader가 세부 작업을 계획합니다
  - Worker들이 인터넷 검색, 데이터 수집, 보고서 작성을 합니다
  - CEO가 결과를 취합해서 최종 보고서를 전달합니다

리서치, 분석, 마케팅, 개발, 법률 등 다양한 분야의 작업이 가능합니다.

====================================================================
  3. 터미널 열기
====================================================================

"터미널"은 컴퓨터에 명령어를 입력하는 검은 화면입니다.
이 프로그램은 터미널에서 실행합니다.

--- Mac의 경우 ---

  방법 1: Spotlight 검색
    1. 키보드에서 Command(⌘) + 스페이스바를 동시에 누릅니다
    2. "터미널" 또는 "terminal" 이라고 입력합니다
    3. "터미널.app"이 나타나면 Enter를 누릅니다

  방법 2: Finder에서 찾기
    1. Finder를 엽니다 (하단 독바의 웃는 얼굴 아이콘)
    2. 상단 메뉴에서 [이동] > [유틸리티]를 클릭합니다
    3. "터미널"을 더블클릭합니다

--- Windows의 경우 ---

  방법 1: 검색
    1. 키보드에서 Windows키를 누릅니다
    2. "cmd" 또는 "명령 프롬프트"라고 입력합니다
    3. "명령 프롬프트"를 클릭합니다

  방법 2: PowerShell (권장)
    1. 키보드에서 Windows키를 누릅니다
    2. "powershell"이라고 입력합니다
    3. "Windows PowerShell"을 클릭합니다

-- 팁 --
  터미널에 명령어를 입력할 때는 한 줄씩 입력하고 Enter를 누릅니다.
  앞에 $ 표시가 있으면, $ 뒤의 내용만 입력하세요.
  ($ 표시는 "여기에 명령어를 입력하세요"라는 표시일 뿐입니다)

  4. 프로그램 다운로드
====================================================================

--- Git이 이미 설치되어 있는 경우 ---

터미널에 아래 명령어를 입력합니다:

    설치 후 위의 git clone 명령어를 다시 입력합니다.

  Windows:
    1. https://git-scm.com 에 접속합니다
    2. "Download for Windows"를 클릭합니다
    3. 다운로드된 파일을 실행하고 "Next"만 계속 눌러 설치합니다
    4. 터미널을 닫았다가 다시 열고, git clone 명령어를 입력합니다

--- Git 없이 직접 다운로드 ---

  1. 웹 브라우저에서 https://github.com/JunyoungOh/AI_Team 접속
  2. 초록색 "Code" 버튼 클릭
  3. "Download ZIP" 클릭
  4. 다운로드된 ZIP 파일을 바탕화면에 압축 해제
  5. 폴더 이름을 "Langgraph"로 변경

  5. Python 설치
====================================================================

이 프로그램은 Python 3.11 이상이 필요합니다.

--- 이미 설치되어 있는지 확인 ---

  $ python3 --version

  "Python 3.11.x" 이상이 나오면 이 단계를 건너뛰세요.
  "command not found" 또는 3.10 이하가 나오면 아래를 진행합니다.

--- Mac 설치 ---

  1. https://www.python.org/downloads/ 접속
  2. 노란색 "Download Python 3.13.x" 버튼 클릭
  3. 다운로드된 .pkg 파일 더블클릭
  4. "계속" -> "동의" -> "설치" 순서로 진행
  5. 터미널을 닫았다가 다시 열고 확인:
     $ python3 --version

--- Windows 설치 ---

  1. https://www.python.org/downloads/ 접속
  2. 노란색 "Download Python 3.13.x" 버튼 클릭
  3. 다운로드된 .exe 파일 실행
  4. ★★★ 중요: "Add Python to PATH" 체크박스를 반드시 체크! ★★★
  5. "Install Now" 클릭
  6. 터미널을 닫았다가 다시 열고 확인:
     $ python --version

====================================================================
  9. API 키 설정
====================================================================

이 프로그램은 Claude Code CLI의 로그인 세션을 그대로 사용하기 때문에
Anthropic API 키는 필요 없습니다. 추가로 필요한 키는 딱 두 개입니다.

  [1] LAW_OC           - 법령상담 모드에서 국가법령정보 조회용 (필수)
  [2] FIRECRAWL_API_KEY - 웹 페이지 스크래핑 품질 향상 (권장)

나머지 기능(웹 검색, 기업 재무 조회, 통계 데이터 등)은 Claude Code가
내장 웹 검색으로 알아서 처리하므로 별도 키 없이도 동작합니다.

--- 9-1. .env 파일 만들기 ---

프로그램 폴더(~/Desktop/Langgraph) 안에 ".env" 라는 이름의
텍스트 파일을 만듭니다.

  Mac:
  $ cd ~/Desktop/Langgraph
  $ nano .env

  (nano는 터미널 텍스트 편집기입니다)
  아래 내용을 입력한 후 Ctrl+O -> Enter -> Ctrl+X 로 저장합니다.

  Windows:
  $ cd ~/Desktop/Langgraph
  $ notepad .env

  (메모장이 열립니다. 아래 내용을 입력하고 저장합니다.)

--- 9-2. .env 파일 내용 ---

아래 내용을 복사해서 붙여넣고, "여기에_키_입력" 부분을
실제 API 키로 교체합니다.

- - - - - - - - - - - 여기부터 복사 - - - - - - - - - - -

# [필수] 국가법령정보 OC 키 - 법령상담 모드에서 사용
LAW_OC=여기에_키_입력

# [권장] Firecrawl API 키 - 고품질 웹 스크래핑
# 없어도 앱은 동작합니다 (일반 HTTP fetch로 자동 폴백)
FIRECRAWL_API_KEY=여기에_키_입력

- - - - - - - - - - - 여기까지 복사 - - - - - - - - - - -

--- 9-3. API 키 발급 방법 ---

  [1] LAW_OC (국가법령정보 공동활용, 무료):
    1. https://open.law.go.kr/LSO/openApi/guideList.do 접속
    2. "OPEN API 활용신청" 메뉴로 이동
    3. 본인 이메일 앞부분(예: hong@gmail.com -> hong)을
       OC로 신청합니다
    4. 승인까지 며칠이 걸릴 수 있습니다 (담당자 수동 검토)
    5. 승인 후 발급된 OC 값을 LAW_OC=... 뒤에 붙여넣기

  ※ 법령상담 모드를 당장 쓰지 않을 예정이라면 이 키는 나중에
    추가해도 됩니다. 일단 빈 값으로 두고 설치를 진행하세요.

  [2] Firecrawl (웹 스크래핑, 무료 500회/월):
    1. https://firecrawl.dev 접속
    2. 회원가입 (Google 계정 가능)
    3. Dashboard에서 API Key 복사
    4. FIRECRAWL_API_KEY=... 뒤에 붙여넣기

-- 참고 --
  LAW_OC가 없으면 법령상담 모드만 동작하지 않습니다.
  다른 모드(모의토론, 미래아이디어, Engineering, Agent 등)는
  LAW_OC 없이도 정상 동작합니다.

  Firecrawl 키가 없어도 프로그램은 실행됩니다.
  다만 웹 페이지 내용을 가져올 때 품질이 다소 떨어질 수 있습니다.

  11. 사용 방법
====================================================================

--- 11-1. 작업 지시 예시 ---

  어떤 종류의 작업이든 자연어로 지시할 수 있습니다:

  리서치:
    "2026년 한국 AI 시장 동향을 분석하고 보고서를 작성해줘"
    "테슬라와 현대자동차의 전기차 전략을 비교 분석해줘"

  마케팅:
    "30대 직장인 대상 건강식품 마케팅 전략을 수립해줘"

  개발:
    "Python으로 할일 관리 앱을 만들어줘"

  분석:
    "카카오 2025년 실적을 분석하고 투자 의견을 제시해줘"

--- 11-2. 진행 과정 ---

  작업을 지시하면 아래 단계로 자동 진행됩니다:

  Step 1 >> 작업 접수
  Step 2 >> CEO 도메인 라우팅         (CEO가 적합한 팀을 배정)
  Step 3 >> CEO 질문 생성             (필요시 질문을 합니다)
  Step 4 >> 사용자 답변 접수          (질문에 답변하세요)
  Step 5 >> 작업 분해 및 배정         (세부 작업 계획 수립)
  Step 6 >> CEO 계획 확인             (계획 품질 검토)
  Step 7 >> 실무 작업 실행            (실제 작업 수행 - 가장 오래 걸림)
  Step 8 >> 결과 갭 분석              (결과 품질 검토)
  Step 9 >> CEO 최종 보고서           (최종 보고서 생성)

  전체 소요 시간: 보통 5~20분 (작업 복잡도에 따라 다름)

--- 11-3. 질문에 답변하기 ---

  중간에 AI가 질문을 할 수 있습니다.
  예를 들어:

    -- research 리더 질문 --
    1. 분석 범위는 국내만인가요, 글로벌인가요?
      답변 1: [여기에 입력하고 Enter]

    2. 특별히 중점적으로 볼 기업이 있나요?
      답변 2: [여기에 입력하고 Enter]

  모르겠으면 "알아서 판단해주세요" 라고 입력해도 됩니다.

--- 11-4. 결과 확인 ---

  작업이 완료되면 최종 보고서가 터미널에 출력됩니다.
  파일로 저장된 경우 data/ 폴더에서 확인할 수 있습니다.

--- 11-5. 연속 작업 ---

  첫 번째 작업이 끝나면 아래 메시지가 나옵니다:

    새 작업을 입력하거나, 종료하려면 'q'를 입력하세요:

  새로운 작업을 바로 입력하거나, q를 눌러 종료합니다.

====================================================================
  13. 자주 묻는 질문 (FAQ)
====================================================================

Q: "command not found: python3" 에러가 나와요.
A: Python이 설치되지 않았습니다. 5번 단계를 따라 설치하세요.
   Windows에서는 python3 대신 python 을 사용해보세요.

Q: "No module named 'src'" 에러가 나와요.
A: 프로그램 폴더 안에 있지 않습니다. 아래 명령어로 이동하세요:
   $ cd ~/Desktop/Langgraph

Q: "pip: command not found" 에러가 나와요.
A: python3 -m pip install -e . 을 시도해보세요.

Q: 법령상담 모드에서 "LAW_OC 설정 안 됨" 에러가 나와요.
A: 9번 단계에서 LAW_OC 키를 .env에 추가했는지 확인하세요.
   국가법령정보(https://open.law.go.kr)에서 무료로 발급되며,
   승인까지 며칠이 걸릴 수 있습니다.

Q: 작업이 너무 오래 걸려요.
A: 복잡한 작업은 10~20분 걸릴 수 있습니다.
   진행률 바가 표시되니 기다려주세요.
   30분 이상이면 Ctrl+C로 중단하고 다시 시도하세요.

Q: "Claude Code" 관련 에러가 나와요.
A: Claude Code CLI가 설치되어 있는지 확인하세요:
   $ claude --version
   로그인이 되어있는지도 확인하세요:
   $ claude (실행 후 로그인 절차 진행)

Q: API 비용이 얼마나 드나요?
A: Claude Code CLI의 로그인 세션을 사용하므로 Claude 사용량에
   따라 과금됩니다.
   간단한 작업: 약 $0.5~2
   복잡한 리서치: 약 $3~10
   Claude Max Plan (월정액)을 사용하면 월 고정 비용입니다.
   LAW_OC와 Firecrawl(무료 500회/월)은 추가 비용이 없습니다.

Q: 프로그램을 업데이트하고 싶어요.
A: 프로그램 폴더에서:
   $ cd ~/Desktop/Langgraph
   $ git pull
   $ pip3 install -e .

Q: Windows에서 한글이 깨져요.
A: PowerShell에서 아래 명령어를 먼저 실행하세요:
   $ chcp 65001

Q: 동시에 여러 작업을 할 수 있나요?
A: 터미널 창을 여러 개 열면 각각 독립적으로 실행할 수 있습니다.
   다만 API 비용이 동시에 발생합니다.
